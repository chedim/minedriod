Welcome to MineDroid documentation website

MineDroid is a port of Android components for Minecraft. It makes GUI creation simplier and easier. It takes care about all of your needs, providing you:

  • Ability to edit and store your interfaces as XML layout files;
  • Basic widgets: TextView, ImageView;
  • Powerful drawing tools with Drawable classes;
  • Layout components: LinearLayout and RelativeLayout;
  • Content scrolling with ScrollView component;
  • Lists rendering with ListView component;

Downloading and installing

You can download a separate jar or checkout the sources from github.
