Welcome to MineDroid documentation website

MineDroid is a port of Android components for Minecraft. It makes GUI creation simplier and easier. It takes care about all of your needs, providing you:

  • Ability to edit and store your interfaces as XML layout files;
  • Basic widgets: TextView, ImageView;
  • Powerful drawing tools with Drawable classes;
  • Layout components: LinearLayout and RelativeLayout;
  • Content scrolling with ScrollView component;
  • Lists rendering with ListView component;

Downloading and installing

You can download a separate jar or checkout sources from github.
MineDroid comes as a separate Forge mod so you don't need to include it into your module jar file.

If you use Gradle, you can add Minedroid from a GitHub hosted maven repository: 

https://raw.github.com/chedim/maven-repo/master/

MineDroid artifact is: 
com.onkiup.minedroid:minedroid:1.0

Also there is a plugin for Gradle that generates R class with links to your mod all resources: 
com.onkiup.gradle:minedroid:1.1-SNAPSHOT

Here is an example gradle.build file:
buildscript { repositories { maven { name = "forge" url = "http://files.minecraftforge.net/maven" } maven { name="onkiup" url = "https://raw.github.com/chedim/maven-repo/master/"; } } dependencies { classpath 'net.minecraftforge.gradle:ForgeGradle:1.2-SNAPSHOT' classpath 'com.onkiup.gradle:minedroid:1.1-SNAPSHOT' } } apply plugin: 'forge' apply plugin: 'minedroid' dependencies { compile 'com.onkiup.minedroid:minedroid:1.0' }
