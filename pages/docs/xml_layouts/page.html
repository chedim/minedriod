XML Layouts

XML layouts allow you to edit and store your screen layouts in XML files. Every tag in layout file represents a View (or Drawable). To access tags attributes, MineDroid uses namespace 'http://onkiup.com/minecraft/xml'.

Attributes

View

  • id
  • width, height — dimen
  • margin, margin-left, margin-top, margin-right, margin-bottom — dimen
  • padding, madding-left, padding-top, padding-right, padding-bottom - dimen
  • background — drawable

ContentView

  • horizontal-gravity — Gravity constants: left, center, right
  • vertical-gravity — Gravity constants: top, center, bottom

TextView

  • text — string

LinearLayou

  • orientation — orientation constant: vertical or horizontal

RelativeLayout children layout attributes

  • alignLeft, alignRight, toRightOf, alignCenter, alignRight, toRightOf, alignTop, below, alignMiddle, alignBottom, above — ids

ListView

  • empty — empty list drawable location
