Resource Files

MineDroid organizes and handles almost all of your mod resources and implements some new resources types.

MineDroid Resources Types

  • layouts/ — layout XML files
  • drawables/ — Drawables XML files or images
  • textures/ — Textures files (not realized yet)
  • ninepatches/ — NinePatch folders
  • raw — Other resources
  • values/strings.xml — String values
  • values/arrays.xml — Array values
  • values/plurals.xml — Plural values
  • values/dimen.xml — Dimension values
  • values/color.xml — Color values

Qualifiers

Minedroid allows you to add some qualifiers to resource or value directories names to differentiate the look of your mod for different environments and localize it. Here is a list of available qualifiers:

  • -[version] — Minedroid version qualifier. Qualifier should look like '-1.7' or '-1.8.3' (or any Minecraft version, preceded with minus symbol). Version 1.8.3 will match, i.e., qualifiers -1.8.3, -1.8 and, -1.
  • -[graphics] — Graphics setting qualifier. Can be either '-fast' or '-fancy'
  • -[mode] — Server mode: -local for single player on integrated server, -remote for multiplayer on dedicated server
  • -[language] — 2-symbol Minecraft client language option. Should look like '-en' or '-ru', etc.

Qualifiers should be added to a resources or values directory name in this order: version, graphics, mode, language: 

values-1.8-fancy-remote-ru

These examples of qualifiers usage are incorrect: 

values-ru-fancy
values-remote-1.8
values-ru-1.8
